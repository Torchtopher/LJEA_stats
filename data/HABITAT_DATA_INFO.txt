LJEA Habitat Data Documentation
================================

Files:
- LJEA_habitat_data.xlsx - Summary scores and site details
- LJEA_HabitatData.csv - Detailed assessment scores with all components

Excel File Structure (LJEA_habitat_data.xlsx):
----------------------------------------------
Sheet "Summary":
- Row 3: Headers (Location, Lat, Long, 2018, 2019, 2020, 2021, 2022, 2023, 2024, 2025, Count, Average)
- Rows 4+: Site data with Total Scores per year
- Column 13 (Average): Mean score across all years
- Columns 14-15: Pre/Post Helene NCBI scores (for some sites)

Sheet "Details":
- Site metadata (Site ID, Name, Lat, Long, Team, Notes)

CSV File Structure (LJEA_HabitatData.csv):
------------------------------------------
Detailed assessment data starting at row 34 (after header info)
Key columns:
- Observation Date: YYYYMMDD format
- Observation Site: Site name
- Habitat Score: Total Score (0-100)
- BEHI Score: Bank Erosion Hazard Index

Habitat Score Components (each scored 0-20 or similar):
- Channel Modification
- Instream Habitat
- Bottom Substrate
- Pool Variety
- Riffle Habitats
- Erosion
- Vegetation
- Light Penetration
- Left Bank (stability)
- Right Bank (stability)

Total Score Interpretation:
- 80-100: Excellent habitat
- 60-79: Good habitat
- 40-59: Fair habitat
- 20-39: Poor habitat
- 0-19: Very poor habitat

Parsing Notes:
--------------
- For Summary sheet: skip first 3 rows, use row 3 as header
- For CSV detailed data: data starts at row 34 (0-indexed: 33)
- Date format in CSV: YYYYMMDD (parse with pd.to_datetime format='%Y%m%d')
- Some sites have "(LJ##)" suffix indicating LJEA site number
